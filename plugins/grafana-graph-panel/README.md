# Graph Panel

Webpack copy of [Grafana default panel](http://docs.grafana.org/features/panels/graph/). 
Now you can make your plugins based on graph panel and hack it easier.

Works only on Grafana versions >= V5.0.1 

# Build

```
npm install
npm run build
```
